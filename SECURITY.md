## Reporting a Vulnerability

We take the security of our software seriously. If you suspect that you have found a security vulnerability in the **Classic Cocktails & Signatures API**, please report it to us immediately.

### How to Report

**Do not open a GitHub issue or pull request for security vulnerabilities.**

Instead, please email us at [contact@aristotelis.bar](mailto:contact@aristotelis.bar) with the following information:

*   A description of the vulnerability.
*   Steps to reproduce the issue.
*   Any relevant logs or output.

We will acknowledge your report within 48 hours and provide a timeline for a fix.

### Disclosure Policy

We ask that you give us a reasonable amount of time to fix the vulnerability before disclosing it to the public. We will notify you when the fix has been released.

### Recognition

We appreciate the efforts of security researchers in helping us keep our software safe. We will acknowledge your contribution in our release notes (unless you wish to remain anonymous).
